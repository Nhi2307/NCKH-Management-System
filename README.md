# Hệ Thống Quản Lý Đề Tài NCKH Sinh Viên

## Mô Tả
Hệ thống quản lý toàn quy trình đề tài nghiên cứu khoa học sinh viên (NCKH-SV) từ đăng ký đến nghiệm thu.

## Tính Năng Chính
- ✅ Đăng ký đề tài (cá nhân hoặc nhóm)
- ✅ Quản lý hồ sơ và duyệt đề tài
- ✅ Phân công giảng viên hướng dẫn
- ✅ Theo dõi tiến độ thực hiện
- ✅ Quản lý hội đồng nghiệm thu
- ✅ Báo cáo và thống kê

## Công Nghệ
- Node.js + Express.js
- SQLite / PostgreSQL
- Bootstrap 5
- JWT Authentication

## Cài Đặt
```bash
npm install
npm start
```
